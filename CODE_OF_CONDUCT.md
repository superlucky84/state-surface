# Code of Conduct

## Our Commitment

We want StateSurface to be a respectful, inclusive, and harassment-free community for everyone, regardless of age, body size, disability, ethnicity, sex characteristics, gender identity and expression, level of experience, education, socioeconomic status, nationality, personal appearance, race, religion, or sexual identity and orientation.

We commit to acting and interacting in ways that build an open and welcoming environment.

## Expected Behavior

- Use respectful, constructive language.
- Assume good intent and discuss technical disagreement with evidence.
- Provide clear, reproducible context when reporting problems.
- Accept feedback gracefully and focus on improvement.
- Respect project boundaries, maintainers' time, and review decisions.

## Unacceptable Behavior

- Harassment, discrimination, insults, or personal attacks.
- Trolling, inflammatory comments, or sustained disruption.
- Publishing others' private information without explicit permission.
- Sexualized language or imagery in project spaces.
- Any conduct that is inappropriate in a professional environment.

## Scope

This Code of Conduct applies to all project spaces, including issues, pull requests, discussions, and any public or private communication conducted in the context of this project.

## Enforcement

Maintainers are responsible for clarifying and enforcing this Code of Conduct. They may remove, edit, or reject comments, commits, code, issues, and other contributions that violate this policy.

## Reporting

Report incidents to the maintainers via GitHub (open an issue labeled `conduct` or contact the repository owner directly). Reports will be reviewed promptly and handled as confidentially as possible.

## Enforcement Guidelines

Maintainers may apply actions based on severity and pattern:

1. Warning for minor, first-time violations.
2. Temporary suspension from participation for repeated or serious violations.
3. Permanent ban for severe harassment, threats, or repeated violations.
